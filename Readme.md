# Edusustain 
EduSustain is a platform designed to foster a culture of sustainability within educational institutions. By integrating sustainability-focused collaboration tools, it enables students, educators, and institutions to work together in promoting eco-friendly practices and integrating sustainability into academic processes.
## Features
- Sustainability Challenges: Engage students in real-world sustainability challenges and projects.

- Collaboration Hub: Connect with educators, researchers, and industry professionals.

- Gamification & Rewards: Encourage participation through leaderboards and incentives.

- Resource Library: Access sustainability-related courses, research papers, and case studies.

- Data Analytics: Track sustainability efforts and measure impact.

## Technologies Used
- HTML
- CSS
- JS
- MongoDB
